# Alaya Protocol v0.3 — Three Natures (三性) Design

## Outcome

Deepen the Yogacara-inspired engineering model by adding epistemological stratification:
seeds are now classified by epistemic reliability — speculative (遍计所执性),
conditional (依他起性), or principle (圆成实性) — with different evidence
requirements, decay rates, and activation weights per nature.

## Nature Types

| Nature | Meaning | Default? | Auto-promotion possible? |
|--------|---------|----------|------------------------|
| `speculative` | Unverified single-source model output | Yes | → conditional |
| `conditional` | Multi-channel repeated pattern | No | → principle |
| `principle` | Cross-domain stable pattern, zero counterexamples | No | No |

## Behavior Matrix

| Behavior | speculative | conditional | principle |
|----------|------------|-------------|-----------|
| Default for new seeds | Yes | — | — |
| Supports to promote | 3 | 2 (to next level: 5) | — |
| Contradiction loss | -0.25 | -0.20 | -0.15 |
| Half-life (days) | 90 | 180 | 365 |
| Decay can deprecate | Yes | Yes | No |
| Contradictions to demote | 1 | 1 (to spec.) | 3 (to conditional) |
| Activation confidence weight | 0.2 | 0.3 | 0.4 |
| Counterexample penalty | 0.2 | 0.3 | 0.5 |

## Promotion/Demotion Rules

```
speculative ──(3 supports)──→ conditional ──(5 supports+0 contradict+30d active)──→ principle
speculative ←──(1 contradict)── conditional ←──(3+ contradicts)── principle
any → deprecated: confidence < deprecate_below (except principle: cannot be deprecated by decay)
```

## Policy Configuration

New fields on `EvolutionPolicy`:

```
speculative_promotion: int = 3
conditional_promotion: int = 2  
principle_auto_promotion_supports: int = 5
principle_min_contradictions: int = 3
speculative_contradiction_loss: float = 0.25
principle_contradiction_loss: float = 0.15
speculative_half_life: float = 90.0
principle_half_life: float = 365.0
```

## Data Model Change

`ExperienceSeed` gains `nature: Nature = "speculative"` field. Schema version stays 1.0;
nature is additive (defaults to "speculative" on import from old payloads).

## Files Changed

| File | Changes |
|------|---------|
| `models.py` | +Nature type, +nature field on ExperienceSeed |
| `engine.py` | +nature-aware reinforce/decay/activate, policy params |
| `cli.py` | +plant --nature flag |
| `__init__.py` | Export Nature type |
| `tests/` | Nature-specific promotion/demotion/activation tests |

## Backward Compatibility

- All existing seeds default to `nature="speculative"` on import
- Existing API unchanged; `ExperienceSeed.new()` defaults to speculative
- PolicyProtocol forward-compatible (new methods have defaults)
