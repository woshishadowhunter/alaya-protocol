# Contributing

Thank you for improving portable, auditable agent experience.

1. Open an issue describing the user problem and expected behavior.
2. Add or update a failing test before implementation.
3. Keep the core dependency-light and framework-neutral.
4. Run `PYTHONPATH=src python -m unittest discover -s tests -v`.
5. Explain privacy, compatibility, or schema implications in the pull request.

Good first contributions include additional deterministic evaluations, documentation translations, protocol examples in other languages, and adapters maintained outside the core package.
